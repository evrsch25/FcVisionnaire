# Images des badges

## Structure

```
public/badges/{type}/{tier-file}.png
```

| Tier code (app) | Fichier sur disque |
|-----------------|-------------------|
| bronze | `bronze.png` |
| silver | `silver.png` |
| gold | `gold.png` |
| legend | `platinium.png` |

Types : `girouette`, `visionnaire`, `sniper`, `chat-noir`, `touriste`, `parachute`

## Barème

| Badge | Bronze | Argent | Or | Légende |
|-------|--------|--------|-----|---------|
| Girouette | 3 paris girouette d'affilée | 6 | 9 | 10+ |
| Visionnaire | 3 gros scores exacts (4+ buts) | 6 | 9 | 10+ |
| Sniper | 2× « 2 scores exacts d'affilée » | 4× | 6× | 7×+ |
| Chat Noir | 3 matchs à 0 pt d'affilée | 6 | 9 | 10+ |
| Touriste | dernier 3 recalculs d'affilée | 6 | 9 | 10+ |
| Parachute | −3 places (≥3) cumulé 3× | 6× | 9× | 10×+ |
